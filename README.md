## 💼 简历亮点（STAR 法则）

### 项目一：电商用户价值分析
> **独立完成基于 MySQL + PowerBI 的电商全链路数据分析，构建 RFM 用户分层模型，识别高价值用户群体（占比 8%，贡献 30% GMV），交付 4 页交互式销售运营看板。**

### 项目二：App 增长与 A/B 测试分析
> **基于 MySQL + Python + PowerBI 完成 App 市场增长分析，构建四级转化漏斗并设计 A/B 实验，使用 Welch's t 检验验证 UI 改版使安装转化率提升 12.3%（p<0.05），全流程复现大厂数据实验范式。**
